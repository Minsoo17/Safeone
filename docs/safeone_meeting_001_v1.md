## 1. 프로젝트 개요

### 1.1 서비스명
- **세이프원 (SafeOne)** - 가칭

### 1.2 핵심 정의
> 안전업체와 안전요원(구직자)을 연결하는 매칭 플랫폼

### 1.3 시장 구조 이해

**현재 계약 흐름:**
```
학교 ──(학교장터/나라장터)──→ 안전업체 ──(네이버 카페)──→ 안전요원
```

**핵심 인사이트:**
- 학교는 플랫폼에 직접 접근하지 않음 (공공조달 시스템 이용)
- 세이프원의 타겟은 **안전업체 ↔ 안전요원** 구간

---

## 2. 벤치마킹: (주)디스탑안전교육

### 2.1 기업 개요
| 항목 | 내용 |
|------|------|
| 대표 | 전선희 (대한민국 수학여행안전요원 1호) |
| 설립 | 2014년 (세월호 참사 이후) |
| 실적 | 2,000여 건 안전활동, 120여 명 안전요원 보유 |
| 연락처 | 1666-1724 |

### 2.2 현재 구인 방식
1. 네이버 카페에 구인글 게시
2. 선착순 댓글 접수
3. 개별 연락 → 자격 검증 → 계약서 작성

### 2.3 현재 방식의 문제점
- 선착순 방식으로 인력 품질 검증 어려움
- 매칭 과정이 수동적이고 시간 소요
- 이력/평판 관리 시스템 부재
- 노쇼 및 부적격 인력 리스크

## 3. 수익 모델 설계

### 3.1 기본 원칙
> "안전요원은 무료, 안전업체에서 수익 창출"

**논리적 근거:**
- 안전요원 유료화 시 가입 욕구 저하
- 플랫폼 핵심 자산 = 인력풀 (양면시장의 보조금 측)
- 안전업체의 기존 업무(카페 관리, 개별 연락, 계약서 작성)를 대체하면 비용 지불 의향 발생

### 3.2 단계별 수익 모델

```
[1단계] 안전요원 무료 가입 → 인력풀 확보 (임계치: 500~1,000명)
            ↓
[2단계] 안전업체 유료화 → 핵심 매출원
            ↓
[3단계] 안전요원 프리미엄 → 부가 매출
            ↓
[4단계] 보험사 제휴 → 수수료 + 우회 방지 + 신뢰도
            ↓
[5단계] 데이터 사업 → 리스크 분석, 보험 상품 개발 협업
```

### 3.3 안전업체 대상 수익 모델 옵션

| 모델 | 구조 | 장점 | 단점 |
|------|------|------|------|
| 건당 수수료 | 매칭 성사 시 일당의 5~10% | 성과 기반, 부담 낮음 | 매칭 우회 가능성 |
| 월정액 구독 | 월 5~10만원 정액제 | 안정적, 예측 가능 | 소규모 업체 진입장벽 |
| 하이브리드 | 기본료 + 건당 수수료 | 안정성 + 성장성 | 구조 복잡 |
| 프리미엄 기능 | 기본 무료 + 상위노출 유료 | 진입장벽 낮음 | 초기 수익 적음 |

### 3.4 매칭 우회(탈플랫폼) 문제

**문제 정의:**
```
[1회차] 플랫폼 통해 매칭 → 수수료 지불
[2회차] "이 사람 괜찮네" → 직접 연락 → 수수료 0원
```

**우회 방지 전략:**
| 전략 | 방식 | 효과 |
|------|------|------|
| 계약/정산 플랫폼化 | 계약서, 급여 정산을 플랫폼에서만 | 우회 시 행정 불편 |
| 평가 기록 귀속 | 플랫폼 통해야만 평가 누적 | 안전요원 이용 유인 |
| 월정액 병행 | 건당이 아닌 구독료 중심 | 우회해도 이미 비용 지불 |
| 보험/보증 연계 | 플랫폼 매칭 건만 보험 적용 | 업체 리스크 관리 |

### 3.5 안전요원 대상 부가 수익 (후순위)

| 기능 | 설명 | 예상 가격 |
|------|------|-----------|
| 프로필 상위 노출 | 업체 검색 시 우선 표시 | 월 1~2만원 |
| 프리미엄 뱃지 | "인증 전문가" 마크 부여 | 월 5천~1만원 |
| 일감 알림 우선권 | 신규 공고 먼저 알림 | 건당 or 월정액 |
| 이력서 강화 | 영상 소개, 포트폴리오 | 1회성 or 구독 |

## 4. 핵심 기능: 평가 시스템

### 4.1 평가 시스템의 가치

**안전업체의 현재 딜레마:**
- 자격증은 확인 가능 → 실제 현장 역량은?
- "써봐야 안다" → 리스크를 업체가 떠안음
- 문제 발생 시 → 학교와의 신뢰 손상

**세이프원 해결책:**
- 누적 평가 데이터로 검증된 인력 필터링
- "평점 4.5 이상, 파견 10회 이상" 조건 검색 가능

### 4.2 안전요원 프로필 구조 (안)

```
[안전요원 프로필]
├─ 기본 자격: 적십자 현장체험학습안전과정 수료 등
├─ 누적 활동: 파견 횟수, 주간/야간 경력
├─ 평가 기록: 업체별 평점, 코멘트
│    - 시간 엄수 ★★★★★
│    - 돌발상황 대처 ★★★★☆
│    - 학생 소통 ★★★★★
└─ 신뢰 등급: 골드 / 실버 / 일반
```

### 4.3 양방향 평가 시스템

| 평가 주체 | 평가 대상 | 평가 항목 예시 |
|-----------|-----------|----------------|
| 안전업체 → 안전요원 | 시간 준수, 현장 태도, 전문성, 재고용 의향 |
| 안전요원 → 안전업체 | 정산 정확성, 소통, 근무환경 안내 |

**효과:** 좋은 업체에 좋은 인력이 몰리는 선순환

## 5. 사업 확장 가능성: 보험 연계

### 5.1 보험사 유치 논리

**보험사 입장에서 세이프원의 가치:**
- 데이터 기반 리스크 평가 (평가 기록, 사고 이력)
- 단체 계약 효율 (개별 영업 불필요)
- 신규 시장 접근 (현장체험학습 안전 분야)

### 5.2 보험 연계 수익 구조

1. **보험료 중개 수수료**: 가입 시 10~20%
2. **프리미엄 구독 포함**: 매칭 = 기본 보험 자동 적용
3. **데이터 라이선싱**: 익명화된 리스크 데이터 판매

### 5.3 전략적 효과

- 우회 방지: 플랫폼 매칭 건만 보험 적용
- 신뢰도 강화: 학교 입장에서 보상 체계 명확
- 포지셔닝: "단순 매칭 플랫폼" → "안전요원 생태계 인프라"
